java设计模式

发表于 2022-06-04

设计模式代表有经验的面向对象软件开发人员使用的最佳实践。设计模式是软件开发人员在软件开发过程中面临的一般问题的解决方案。

设计模式的类型

3类23种

编号 模式和说明
1 创造模式 –创建对象而隐藏创建逻辑的方法，不是直接使用new来实例化对象
2 结构模式 –设计类和对象组成，继承概念用于组成接口并定义方法以获得新功能
3 行为模式 –侧重于对象之间的通信
4 J2EE模式 –设计表示层 由Sun Java Center表示

在核心Java种主要有3种设计模式
